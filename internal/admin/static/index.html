配置接口
GET/api/config
说明：读取完整配置。
请求参数：无。响应参数：`port`、`log_request_params`、`routes`、`tcp_routes`、`ssh_tunnels`。
PUT/api/config
说明：保存完整配置。
请求参数：完整配置 JSON。响应参数：`status`。
事件接口
GET/api/events
说明：SSE 指标推送，5 秒更新一次。
响应参数：HTTP/TCP 统计、CPU、内存、磁盘指标。
GET/api/logs/events
说明：SSE 日志流。
响应参数：`kind`、`message`、`listen`、`source`、`target`、`bytes`、`duration_ms`。
HTTP 接口
POST/api/routes/add
说明：新增 HTTP 路由。
请求参数：路由对象。响应参数：`status` 或 `error`。
PUT/api/routes/update
说明：更新 HTTP 路由。
请求参数：`original_path`、`route`。响应参数：`status` 或 `error`。
POST/api/routes/delete
说明：删除 HTTP 路由。
请求参数：`path`。响应参数：`status` 或 `error`。
TCP 接口
POST/api/tcp-routes/add
说明：新增 TCP 转发。
请求参数：`listen`、`enabled`、`upstreams`。响应参数：`status` 或 `error`。
PUT/api/tcp-routes/update
说明：更新 TCP 转发。
请求参数：`original_listen`、`route`。响应参数：`status` 或 `error`。
POST/api/tcp-routes/delete
说明：删除 TCP 转发。
请求参数：`listen`。响应参数：`status` 或 `error`。
SSH 接口
POST/api/ssh-tunnels/add
说明：新增 SSH 隧道。
请求参数：隧道对象。响应参数：`status` 或 `error`。
PUT/api/ssh-tunnels/update
说明：更新 SSH 隧道。
请求参数：`original_name`、`tunnel`。响应参数：`status` 或 `error`。
POST/api/ssh-tunnels/delete
说明：删除 SSH 隧道。
请求参数：`name`。响应参数：`status` 或 `error`。
