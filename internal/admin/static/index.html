使用说明

路径匹配：请求路径以配置的路径开头即匹配

去除前缀：开启后，转发时会去掉匹配的路径前缀

示例：路径 /api，目标 http://192.168.1.10:3000

- 去除前缀：/api/users -> http://192.168.1.10:3000/users

- 保留前缀：/api/users -> http://192.168.1.10:3000/api/users
